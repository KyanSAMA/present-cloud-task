# 8 前端UI框架和标准页面设计

## 8.1 任务描述

### 8.1.1 任务介绍

确定界面的设计风格，选择一种前端UI框架。设计两种标准页面：列表页和编辑页。

### 8.1.2 任务要求

- 了解主流的前端UI框架
  - Bootstrap
  - Material Design
  - Ant Design
  - Element UI
- 设计后台管理系统中常用的页面
  - 首页
  - 列表页
  - 编辑页
